Aplicación de Analista de Inventarios

Desarrollada para automatizar tareas en la empresa.

Funciones principales:

  • Extracción de datos desde archivos Excel, como mermas y sobrestock, generando un archivo CSV para su carga en el servidor.
  • Automatización con Selenium para la descarga de diferencias de inventario por sucursal y su unificación para análisis.
  • Cierre automatizado de inventarios por sucursal.

Medidas de seguridad:

  • Codificación de texto en Base64 para protección.
  • Almacenamiento de datos sensibles en un archivo .env.
  • Ofuscación de datos de acceso para mayor seguridad.

Tecnologías utilizadas:

  • Tkinter, Selenium, Pandas, Dotenv
