Presentación

Mi nombre es William Suárez, soy un profesional de informática con un gran interés orientado fuertemente al Desarrollo, Análisis de Datos y Ciberseguridad.

Mi formacion en Técnico en Informático en Ciberseguridad, y mis cursos en Big Data me dieron una base sólida para personalizar soluciones en mis objetivos.

He realizado proyectos muy personalizados para funcionalidades en empresas donde he trabajado.
He desarrollado aplicaciones, scripts y páginas web que han automatizado y optimizado el rendimiento horas de trabajo, y reducido costos en las empresas donde he trabajado. Mi enfoque se centra en comprender las necesidades específicas de cada cliente para ofrecer soluciones que generen un impacto real en sus operaciones.

Cuento con experiencia en FrontEnd, desarrollando aplicaciones web de interfaces de usuario interactivas:

  • ReactJS
  • Neact-Native
  • Html / Css / JS / Bootstrap

En BackEnd tengo experiencia en el desarrollo de APIs robustas y escalables con:
  • Node-Express
  • Fast-Api
  • Flask
  • Django
  • Firebase

En CI/CD estoy familiarizado con el uso de Git para el control de versiones y la colaboración en proyectos.
También he realizado proyecto con integracion de IA, automatizaciones con Selenium, y analisis de datos en Python (Pandas y NumPy).

Aplicación de Analista de Inventarios

Automatización de tareas

Funciones principales:

  • Extracción de datos desde archivos Excel, como mermas y sobrestock, generando un archivo CSV para su carga en el servidor.
  • Automatización con Selenium para la descarga de diferencias de inventario por sucursal y su unificación para análisis.
  • Cierre automatizado de inventarios por sucursal.

Medidas de seguridad:

  • Codificación de texto en Base64 para protección.
  • Almacenamiento de datos sensibles en un archivo .env.
  • Ofuscación de datos de acceso para mayor seguridad.

Tecnologías utilizadas:

  • Python, Tkinter, Selenium, Pandas, Dotenv

Base de Datos

Desarrollos de BBDD para proyectos

Descripcion de la base de datos:

Este proyecto consistió en el diseño y la implementación de una base de datos robusta y eficiente para gestionar los registros de vehículos y permisos de circulación para una municipalidad.

Funcionalidades claves:

  • Almacenamiento y Gestión de Registros de Vehículos.
  • Gestión de Permisos de Circulación.
  • Consultas y Reportes.
  • Integridad de Datos.
  • Funciones y Procedimientos Almacenados.

Tecnologias utilizadas:

  • Implementación: SQL (MariadB)
  • Diseño de la Base de Datos: Modelo Entidad-Relación (MER), Diagrama de Tablas (MERN)

Documentación:

  • Diagrama de tablas (MERN)
  • Scripts SQL para la creación de tablas, inserción de datos (reales y ficticios), modificaciones, funciones, procedimientos y triggers.
  • Consultas SQL para la extracción y análisis de datos.
